# Лабораторная работа №8

Программный фильтр в Linux называется передача потока stderr
(вывод программы в консоль) на stdin другой программы.
Программа gen генерирует поток данных, состоящий из
суперпозиции двух гармонических функций, на который наложен
"шум", состоящий из случайных значений.
Программа osc представляет "осциллоскоп", выводящий график
поступающих сигналов. Каждое четное значение выводится
другим (синим) цветом.
Фильтр 
./gen | ./osc 
запустит показ сгенерированного сигнала.
Задача:
Написать программу-"сглаживатель"-фильтр, уменьшающий шум
сигнала. stdin - входной поток, stdout - сглаженные данные. 
Встроить в цепочку 
./gen | ./filter | ./osc
Сглаживание осуществляется сверткой входного сигнала с
набором коэффициентов сглаживания (в файле PASCAL.txt есть
такие наборы).
Для работы osc нужна библиотека FLTK и класс рисования
графиков функций Cartesian. FLTK есть на каждом компьютере в классе,
Cartesian присутствует в папке с заданием.
