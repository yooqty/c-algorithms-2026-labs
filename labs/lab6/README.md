# Лабораторная работа №6. Выпуклая оболочка на плоскости.

Используя алгоритм QuickHull с предварительной сортировкой по координате X
построить выпуклую оболочку для множества точек, координаты которых
находятся в файле (для примера дан файл IN.txt). В каждой строке файла
задано целое число (номер точки) и вещественные координаты x и y.
Дополнительные указания:
(1) Использовать библиотечную функцию qsort для сортировки;
(2) Написать вспомогательные функции для проверки,
находится ли следующая точка справа или слева
от луча, соединяющего предпоследнюю точку с последней;
(3) Вывод результата (массив с номерами точек) в файл.

(4) Все исходные точки из файла разместить в массиве
структур типа POINT.
