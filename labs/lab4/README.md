# Лабораторная работа №4

Задача - написать 2 приложения, преобразующий текст на русском языке в транслит и обратно.

Программа должна считывать много (до 1000) текстовых файлов, разбивая текст на слова, и для каждого слова подсчитывать количество символов в нем. Выровнять текст по правой границе столбца.
Файлы содержат текст на русском языке в кодировке cp1251 (Win1251) и находятся в каталогах History и Detect.
Вход программы - список файлов, которые необходимо обработать, передается через параметры командной строки.
Выход программы - ФАЙЛЫ, переведенные на транслит.

Алгоритм должен быть однопроходным, без повторного считывания информации из файлов.
